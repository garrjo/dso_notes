Verification Method

Each note includes a SHA-256 hash of its content concatenated with its ISO timestamp. This creates a tamper-evident record.

To verify: concatenate the note's JSON content (without the hash field) with the timestamp, then compute SHA-256.

For additional verification, commit the exported JSON to a public Git repository. Git's commit hashes provide an additional layer of timestamping.
