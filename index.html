Verification & Trust

Local Hash: Each note includes a SHA-256 hash of its content. This proves the content hasn't been modified since creation.

Blockchain Timestamp: Notes timestamped via OpenTimestamps are anchored to the Bitcoin blockchain. This proves the note existed at a specific point in time, independently verifiable by anyone.

How to verify: Download the .ots proof file and verify at opentimestamps.org or with the ots CLI tool.

Git commits: For additional verification, commit your exported JSON to a public Git repo. Git's commit hashes add another timestamp layer.
